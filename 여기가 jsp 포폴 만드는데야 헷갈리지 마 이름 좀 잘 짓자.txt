영화 등록 리뷰 목록 리뷰 조회 홈으로

-----------------------------------------------------


홈 화면 -> 헤더 MOVIE 클릭 시 영화 등록 페이지

MOVIE 등록 후 리뷰쓰기 클릭 시 리뷰 작성 페이지

헤더 REVIEW 클릭 시 영화/리뷰 리스트 페이지
(영화 등록 전에는 리뷰 작성 불가 / 리뷰는 필수 아님)

헤더 LIST/EDIT 클릭 시 등록한 영화/리뷰 조회 화면 


-------------------------------------------------------

MOVIE 테이블

*번호(등록 시 순서 자동생성), 포스터, *영화 제목, 출연 배우, *감독, 국가, 영화 등급, *장르, 개봉 일자 

REVIEW 테이블
*영화 제목, *리뷰 제목, 관람일자, *내 평점, *내용
